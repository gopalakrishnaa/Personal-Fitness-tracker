---
description: How to commit changes to the git repository
---

1. Check which files have changed:
```bash
git status
```

2. Stage all changes (or specify files):
```bash
git add .
```

3. Commit the changes with a descriptive message:
```bash
git commit -m "Your commit message here"
```

4. (Optional) Push to the remote repository:
```bash
git push
```
